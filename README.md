## 📁 Repository 폴더 구조
```
{플랫폼}/{문제_이름}/{코드_이니셜.java}
```

- 💡 예시: `BOJ/방향_전환/Main_jay.java`

## ⚠️ commit 컨벤션

```
{태그}: {클래스 이름(플랫폼_문제 번호_문제 제목)} {커밋 메시지}
```

- 플랫폼에 따라 없는 내용은 생략 가능
- 💡 예시: `✨feat: BOJ_8382_방향_전환 알고리즘 구현`
